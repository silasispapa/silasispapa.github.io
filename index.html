This is a "Sitting Posture Checker" application.

There will be a sitting score and if it reaches 0, an alarm will sound to notify the user.

Click on the canvas to show/hide the volume bar.

Steps

  1. Please sit in front of the camera and make sure your face and shoulder is visible.
  2. Please use in a well-lit environment.
  3. Press "Start" to run the application and make sure that at least 7 points are detected as shown in the picture for best result.

Cautions!

Switching to another tab/ minimising the browser will cause the application to run abnormally.

Leave the tab/browser window open in the background allows the application running smoothly.
